# Hang On To Each Other

Third-party reference shelf for fragile external knowledge that supports Forgotten Industries restoration work.

This area is for manuals, diagrams, photos, forum images, review assets, and other external references that help reconstruct how abandoned or unsupported hardware goes back together.

## Rules

- Preserve source provenance before using the asset in a public-facing document.
- Keep third-party references separate from Forgotten Industries original photography.
- Prefer descriptive archive filenames while recording original local filenames and hashes.
- Record uncertainty directly.
- Treat external media as reference evidence, not ownership evidence.

## Current Reference Sets

- `caselabs-mercury-s8/` - CaseLabs Mercury S8 assembly and construction references.
