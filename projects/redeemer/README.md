# REDEEMER Instagram Recovery Dossier

Project id: `FI-PROJ-005`
Archive status: recovered from Instagram, first-pass catalogue
Evidence window: 2016-03-14 through 2016-05-17

REDEEMER was the Forgotten Industries white NZXT H440 custom liquid-cooled PC documented across the 2016 Instagram run. The recovered posts establish a build sequence from fittings and loop planning through dual-pump assembly, storage selection, RAM and GPU watercooling prep, tubing, leak testing, custom cabling, Windows install, HUE+ lighting, and final game testing.

This file is an evidence map, not a memory replacement. The canonical recovered social records live in `src/social-posts.yml`; the generated post files under `posts/social/` preserve captions, timestamps, media, source links, likes, and comments.

## Primary Spec

Source: [`BFKAtXeBXT8`](../../posts/social/2016-05-08-instagram-20160508-BFKAtXeBXT8-forgotten-industries-redeemer.md)

- Case: NZXT H440 Mid-Tower, white
- Motherboard: ASUS Z170-Deluxe
- CPU: Intel Core i7-6700K
- Memory: Kingston HyperX Predator 16GB 3000MHz DDR4
- GPU: EVGA GeForce GTX 980 Ti 6GB K|NGP|N
- Storage: Intel 535 Series SSD 480GB x2, RAID0
- PSU: Corsair AX1200i Digital ATX 1200W
- Lighting: NZXT HUE+
- CPU block: EK-Supremacy EVO, Nickel CSQ
- RAM cooling: EK-RAM Monarch Module x4; EK-RAM Monarch x4, Nickel
- GPU block: EK-FC980 GTX Ti Classy KPE, Nickel
- GPU backplate: EK-FC980 GTX Ti Classy KPE Backplate, Nickel
- Pumps: EK-D5 Vario Motor x2
- Pump top: EK-D5 Dual Top G1/4 CSQ, Acetal
- Reservoir: EK-RES X3 250, White
- Fittings: EK-ACF Compression 10/16mm x16, Nickel
- Tubing: EK-Tube ZMT Matte Black 10/16mm
- Radiators: EK-CoolStream XE 240mm and EK-CoolStream XE 360mm
- Fans: EK-Vardar F4-120ER 2200rpm x6, White
