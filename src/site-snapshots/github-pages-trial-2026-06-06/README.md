# GitHub Pages Trial Snapshot

Snapshot date: 2026-06-06

This folder preserves the raw HTML GitHub Pages trial surface before the public site was reset for the custom-domain, one-post-at-a-time publishing flow.

Captured pages:

- `index.html`
- `archive.html`
- `inventory.html`
- `social-posts.html`
- `field-log-template.html`
- `about.html`
- `contact.html`

The snapshot intentionally does not duplicate source assets, imported media, Markdown posts, or generated archive data. Links to `assets/`, `posts/`, `src/`, and `dist/` point back to the repository-level copies so the trial surface can be reviewed without creating a second archive.

Preserve this as reference material. Do not treat it as the live public structure.
