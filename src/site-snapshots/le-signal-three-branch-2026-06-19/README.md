# Le Signal Three-Branch Snapshot

Snapshot date: 2026-06-19

This folder preserves the generated public surface after Le Signal was divided
into three distinct branches:

1. _“Recoveries, Restorations, & Le Rédempteur”_ — the blog.
2. ATLAS Reports — AI-assisted end-of-day debriefs.
3. Field Log — original recorder audio preserved as MP3 entries.

Captured surfaces include the home page, Le Signal index, every generated ATLAS
Report page, the recorder-based Field Log page, Live Dispatches, the ATLAS Report
archive/category views, the projects index, core support pages, the sitemap, and
the compiled stylesheet.

The snapshot intentionally does not duplicate source assets, imported media,
audio, Markdown sources, generated archive data, or older site snapshots. Links
to those resources continue to point at the repository-level copies.

Preserve this as reference material. Do not treat it as the live public
structure.
