# Hang On To Each Other

Technical reference manuals rebuilt from third-party sources, forum archaeology, archived media, and local evidence.

This shelf exists because some instructions disappear. When the original manufacturer documentation is missing, defunct, incomplete, or scattered across old forum posts, the archive can preserve a careful reconstruction.

It is also a credit shelf. If a diagram, review, photo, forum post, or fragment helps the archive, the source should stay visible. Give credit and carry the wisdom forward. Credit is the signal; this is visible attribution, not hidden telemetry.
